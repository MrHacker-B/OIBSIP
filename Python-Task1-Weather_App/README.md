# 🌤 Advanced Weather App

A modern desktop Weather Application built using **Python**, **Tkinter**, and the **OpenWeatherMap API**. This application provides real-time weather information along with a 5-day forecast through a simple and user-friendly graphical interface.

## 📌 Features

- 🔍 Search weather by city name
- 🌡 Real-time temperature
- 💧 Humidity
- 🌬 Wind speed
- ☁ Weather condition
- 🌤 Weather icons
- 🌡 Celsius/Fahrenheit toggle
- 📅 5-Day Weather Forecast
- 🌅 Sunrise & Sunset timings
- 🤒 Feels Like temperature
- 📈 Maximum & Minimum temperature
- 📊 Atmospheric pressure
- 👀 Visibility information
- ❌ Error handling for invalid city names and network issues
- 🖥 Simple and responsive Tkinter GUI

## 🛠 Technologies Used

- Python 3
- Tkinter
- Requests
- Pillow (PIL)
- Geocoder
- OpenWeatherMap API

## ⚙ Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/Weather-App.git
```

2. Move into the project folder:

```bash
cd Weather-App
```

3. Install the required libraries:

```bash
pip install -r requirements.txt
```

4. Add your OpenWeatherMap API key to `config.py`:

```python
API_KEY = "YOUR_API_KEY"
BASE_URL = "https://api.openweathermap.org/data/2.5/"
```

5. Run the application:

```bash
python main.py
```
